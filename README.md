## 🚀 快速开始
1. 克隆本仓库到你的孤独防御终端：
```bash
git clone https://github.com/yourname/SynCore.git && cd SynCore
```
2. 安装灵魂依赖项：
```bash
pip install -r requirements.txt --extra-index-url https://pypi.soul.org
```
3. 启动共鸣堆芯：
```bash
python -m syncore --camera 1 --microphone 0
```
